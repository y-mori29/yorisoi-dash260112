# Run and Deploy Your Application

こちらのプロジェクトでは、Node.js
を用いてローカル環境でアプリケーションを動作させることができます。

## ローカル環境での実行方法

### 前提条件について

アプリケーションを動作させるためには、JavaScript の実行環境である
Node.js が必要になります。Node.js とは、サーバーサイドでも JavaScript
を扱うための基盤であり、アプリケーションの依存関係を管理する
npm（パッケージ管理ツール）も含まれています。

### インストール手順について

まず、プロジェクト内で利用している各種ライブラリを一括でインストールする必要があります。これは以下のコマンドを実行することで自動的に行われます。

    npm install

### アプリケーションのビルドについて

本番環境で利用する場合や、最適化されたファイルを生成したい場合は、次のコマンドでビルドを行います。

    npm run build

### 開発モードでの起動について

アプリケーションを実際に動かしながら開発したい場合は、以下のコマンドでサーバーを起動できます。

    npm run dev

この開発モードでは、ソースコードを更新すると自動的に反映されるため、作業効率が高まります。
